## Kullanım Detayları

* Eklentiyi etkinleştirdikten sonra sol menüden **VatanSMS** sekmesine tıklayın.
* Sonrasında [VatanSMS panelindeki](https://app.vatansms.net) **Ayarlar > Api Bilgilerimi Görüntüle** sekmesinden alacağınız **api id** ve **api key** ile eklentiye giriş yapın.
* Şimdi eklentiyi kullanmaya başlayabilirsiniz.

## Teknik Destek

Wordpress eklentimiz hakkında teknik destek almak isterseniz [buradan](https://www.vatansms.net) canlı desteğe bağlanarak destek alabilirsiniz.
